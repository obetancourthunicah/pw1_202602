Encabezados

Los encabezados en html se representan con las etiquetas <h1> a <h6>, siendo <h1> el encabezado de mayor importancia y <h6> el de menor importancia. Estos encabezados se utilizan para organizar el contenido de una página web y mejorar su legibilidad.

Párrafos

Los párrafos en html se representan con la etiqueta <p>. Esta etiqueta se utiliza para delimitar un bloque de texto que forma un párrafo. Los párrafos pueden contener texto, imágenes, enlaces y otros elementos HTML.

Corte de Linea

Para crear un corte de línea dentro de un párrafo, se utiliza la etiqueta <br>. Esta etiqueta no tiene una etiqueta de cierre y se utiliza para insertar un salto de línea. Por ejemplo:
